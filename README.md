# 项目简介

## 核心功能
- 基于 YOLOv8 的压力表检测系统，支持识别 ['base', 'maximum', 'minimum', 'tip'] 四类压力表
- 支持多种输入源检测：单张图片、图片文件夹、视频文件和本地摄像头
- 提供用户界面，可实时显示目标位置、目标总数和置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持批量检测功能，可直接查看所有检测结果
- 支持检测结果保存功能
- 提供多种模型选择选项

## 适用场景
- 工业压力表自动化检测
- 质量控制系统中的仪表读数识别
- 工业生产线上的压力监测
- 设备维护和检测流程
- 数据采集与分析系统

## 技术特点
- 使用 Python 3.9 开发，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等技术栈
- 包含完整数据集（8000 张标注图片）
- 提供训练好的模型及训练结果
- 配备详细的项目运行教程
- 支持 Pycharm+Anaconda 或 VSCode+Anaconda 开发环境
